# 🏠 Rent Calculator

A simple Python program to calculate total rent and split expenses among multiple people.

---

## 📌 Description

This project helps calculate the total living expenses including rent, bills, and other costs, and then divides the total amount equally among all persons.

It is useful for:

* Students sharing rooms
* Flatmates
* Hostel living

---

## 🚀 Features

* Takes input for multiple expenses:

  * Room rent
  * Water bill
  * Electricity bill
  * Furniture rent
  * Groceries
  * Kitchen appliances
  * Washing products
  * Other expenses
* Calculates total rent
* Splits rent equally among all persons

---

## 🧮 How It Works

1. User enters the number of persons
2. User inputs all expense details
3. Program calculates:

   * Total expenses
   * Rent per person

---

## ▶️ How to Run

1. Make sure Python is installed
2. Run the program:

   ```
   python your_file_name.py
   ```
3. Enter the required inputs when prompted

---

## 📷 Example

```
enter number of persons= 3
enter room rent= 9000
enter water bill= 300
enter electricity bill= 600
enter furniture rent= 500
enter groceries= 2000
enter kitchen appliances= 300
enter washing products= 200
enter other expenses= 100

total rent= 13000
rent per person= 4333
```

---

## ⚠️ Limitations

* No input validation (invalid inputs may cause errors)
* Splits rent equally (no custom distribution)
